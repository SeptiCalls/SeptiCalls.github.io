IVF Analyser

This website determines whether the IVF treatment is suitable for an infertile couple based on a few yes-or-no questions. It acts as a first resort where physically meeting an expert is not an option.

This project was made as an entry for SOSC's Hacknight 2022. It won second place in Junior Level.
